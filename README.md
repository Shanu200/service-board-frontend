# Service Board Frontend

Frontend application for the Service Board platform built using Next.js, TypeScript, and Tailwind CSS.

---

# Features

- User Registration
- User Login
- JWT Authentication
- View All Jobs
- Create Jobs
- View Job Details
- Search Jobs
- Book Jobs
- Profile Page
- Delete Own Jobs
- Responsive UI

---

# Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Axios

---

# Setup Instructions

## 1. Clone Repository

```bash
git clone https://github.com/Shanu200/service-board-frontend.git

## 2. Navigate Into Project

```bash
npm install

## 3. Required Environment Variables
Create a file named:
```bash
.env.local

```bash
NEXT_PUBLIC_API_URL=https://service-board-backend-brh7.onrender.com

## 4. Run Instructions
Start development server:

```bash
npm run dev

Frontend runs on:

```bash
http://localhost:3000

## 5. Build For Production

```bash
npm run build
npm start

## 5. Deployment
Frontend deployed using Vercel.
