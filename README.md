# Service Board Frontend

Frontend application for the Service Board platform built using Next.js, TypeScript, and Tailwind CSS.

---

# Features

- User Registration
- User Login
- JWT Authentication
- View All Jobs
- Create Jobs
- View Job Details
- Search Jobs
- Book Jobs
- Profile Page
- Delete Own Jobs
- Responsive UI

---

# Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Axios
